### Micro Twitter Vue

> Based on [vanilla version](https://github.com/igorhalfeld/micro-twitter)

### How to run

```sh
# install vue cli
npm install -g @vue/cli @vue/cli-service-global

# serve files
vue serve

# build project
vue build
```
